EECS 485 (Web Systems)

In this course we completed 5 projects, starting with a static site generated with Jinja html templates, and finishing by creating a search engine using the MapReduce algorithm.

EECS 481 (Software Engineering)

This course taught high-level concepts in software engineering such as test coverage, test automation, debugging automation, and mutation testing. We finished the course by making a code contribution to open-source software.

EECS 484 (Database Management)

Database Management is a course that focuses on integrating SQL into larger codebases. We started by learning the basics of SQL, then began integrating queries into python code later on, finishing the semester by implementing the Grace Hash Join Algorithm with Python & SQL.

EECS 493 (User Interface Development)

UI Development is a course focused on the basics of UI/UX development. We began by learning HTML/CSS and basic Javascript, and ended the semester pitching a prototype of an application that we created in groups of 4.
